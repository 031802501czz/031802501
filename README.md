# 算法原理
- 余弦定理：将每个字的出现次数和本身保存下来，可以用map，在计算文字向量的余弦值。
- 最小编辑距离：长度为L1的字符1到长度为L2的字符2的转变，操作有三种，添加一个字符，删除一个字符，修改一个字符，不断递归迭代会堆栈异常，采用动态规划完成
